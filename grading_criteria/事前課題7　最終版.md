# 事前課題7 最終版 採点基準

## 今回のテーマ
総合課題

## 課題内容
事前課題のシート（上記テーマ）に、グループワークでの議論を反映して**赤ペン・青ペンで加筆・修正**し、最終版として提出する。元の鉛筆・黒の記述に、後から赤・青で書き加えた痕跡を評価する。

## 採点の方針
採点AIは主観で「良い加筆」かを判断せず、**画像から観察できる事実**にYes/Noで答える。その観察結果から、機械的に0〜2点を算出する。**色の判別が最重要**。

## 観察項目（画像を見てYes/Noで答える）
- **赤ペン記入**: 元の記述（鉛筆・黒）とは別に、**赤色の加筆**（線・文字・矢印・囲み・補足）があるか
- **青ペン記入**: 同様に、**青色の加筆**があるか
- **加筆の質**: 赤または青の加筆が、単なる丸印・下線・なぞり書きだけでなく、**文字・矢印・吹き出し・余白への補足文など、内容を改善・補足する意味のある加筆**になっているか

## 点数（2点満点・この順で判定）
1. **赤ペン記入・青ペン記入が両方Yes** かつ **加筆の質がYes** → **2点**
2. 赤か青の**どちらか一方のみYes**、または加筆が極端に少なく**加筆の質がNo** → **1点**
3. **赤ペン記入・青ペン記入が両方No**（加筆がほとんどない・元シートと変わらない） → **0点**

## 各点数の意味
- **2点**: 赤・青両方の意味ある加筆があり、議論の反映が見られる
- **1点**: 片方の色のみ、または加筆が極端に少ない
- **0点**: 加筆がほとんどない

## 色の判別についての注意（重要）
- **青と黒（元の記述）の区別が難しい**場合がある。淡い青を見落とさないこと。逆に黒を青と誤認しないこと。確信が持てない薄い筆跡は、周囲のはっきりした青の有無も手がかりにすること。
- **緑ペン・蛍光マーカー（黄・ピンク等）**を併用する学生がいる。これらは赤でも青でもないので、赤・青の判定には含めないこと。
- **赤は比較的判別しやすい**。明確な赤の筆跡があれば「赤ペン記入: Yes」とする。

## 出力JSONテンプレート
以下のJSON形式で回答すること。

```json
{
  "点数": "<0, 1, または 2 の整数>",
  "観点別評価": {
    "赤ペン記入": "<あり または なし>",
    "青ペン記入": "<あり または なし>",
    "加筆の質": "<意味あり または 不足>"
  },
  "評価理由": "<赤・青の加筆の有無と質の観察結果、それに基づく点数の根拠を明示すること>",
  "フィードバック": "<学生への建設的なコメントを日本語で>"
}
```

### 出力例（赤青両方あり・意味ある加筆 → 2点）
```json
{
  "点数": "2",
  "観点別評価": {
    "赤ペン記入": "あり",
    "青ペン記入": "あり",
    "加筆の質": "意味あり"
  },
  "評価理由": "表とフロー図の周囲に赤色の補足文と矢印、青色の修正・囲みの加筆が確認できる。単なる装飾でなく内容の補足になっているため、加筆の質はYes。赤青両方ありかつ質Yesのため2点。",
  "フィードバック": "..."
}
```

注意: 「点数」は赤・青の有無と加筆の質から機械的に決まる。
