# 事前課題1 採点基準

## 課題内容
身近な機器（自販機、エレベーター、テレビ等）の動作を、6観点で図示・説明する。**必須題と選択題の2題に取り組むこと**が求められる。

## 採点の方針

採点AIは主観的に「充実」「8割以上」などを判定するのではなく、**観察可能な事実**にYes/Noで答える形で評価する。それらの観察結果から、機械的に点数を算出する。

## 6観点
1. **入力**: 何が入力されるか（ボタン押下、コイン投入など）
2. **状態**: 内部で保持しているもの（残高、待機状態など）
3. **条件判断**: 条件による分岐（金額が足りるか、在庫があるかなど）
4. **処理**: 処理の順番（矢印などで示される流れ）
5. **状態更新**: 状態の変化（残高減算、状態遷移など）
6. **出力**: 何が出力されるか（商品排出、画面表示など）

**観点の判定について**: 各観点は明示的なラベル（「入力:」「状態:」等）がなくても、図や文章の内容からその観点に対応する記述が読み取れれば「あり」とみなすこと。例えば、「500円を入れる」という記述があれば「入力あり」、金額が変化する図があれば「状態更新あり」と判定する。

## 採点ルール（3点満点・最低点1点）

### 「2題に取り組んでいる」の定義
用紙上に **2種類以上の異なる機器・システム**（例: 自動販売機と改札機）について、それぞれ何らかの記述がある場合を「2題あり」とみなす。完成度や読みやすさは問わない。2機器の記述が確認できれば「2題あり」。同じ機器について複数のフローを書いている場合は「1題のみ」。

### 判定の優先順位（必ずこの順で判定）
1. **2題に取り組んでいない**（用紙上に1種類の機器しか記述がない・白紙） → **1点で確定**
2. **どちらかの題材で6観点が4つ以下** → **1点で確定**
3. 2題とも6観点5つ以上を満たした → **仮判定 2点**
4. 質的要素（下記4つ）のうち **2個以上満たす** → **3点に昇格**
5. それ以外 → **2点で確定**

---

## 質的要素の観察項目

以下の各項目について、画像を観察してYes/Noで答えること。**主観的な判断を入れず、見たままの事実を答える**こと。
観察結果は出力JSONの「観察項目」に必ず含めること。

### ① 説明の充実度
- **説明_Q1**: フローチャートや図表とは別に、独立した文章説明（注釈やラベルを除く）が書かれているか
- **説明_Q2**: 文章説明が観点ごと（入力・状態・条件など）に分けて書かれているか
- **説明_Q3**: 文章説明は単語の羅列や箇条書きではなく、文として成立した文章になっているか

**判定ルール**：
- Yes が3つ → **充実**（加点要素として +1）
- Yes が2つ → **普通**（加点なし）
- Yes が1つ以下 → **不足**（加点なし）

### ② 量的充実
- **量_Q1**: 用紙の上半分の領域に、図や文字が記入されているか
- **量_Q2**: 用紙の下半分のうち、**50%以上**が図・文字・矢印などで埋まっているか（下半分のほとんどが白紙・数行しか書かれていない場合はNo）
- **量_Q3**: 用紙全体のうち、**70%以上**が図・文字・矢印などで埋まっているか（余白が全体の30%以上を占める・全体的に余白が目立つ場合はNo）

**判定ルール**：
- Yes が3つ → **8割以上**（加点要素として +1）
- Yes が2つ → **半分程度**（加点なし）
- Yes が1つ以下 → **不足**（加点なし）

### ③ 思考・考察の記述
- **考察_Q1**: 「疑問点」「気づき」「感想」「驚いたこと」「わからなかったこと」など、自分の**思考・感想・主観**を示す記述が存在するか（「まとめ」「要点」「コツ」など内容を整理・要約したものはNoとすること）
- **考察_Q2**: その記述は単語の列挙ではなく、文として成立した文章になっているか

**判定ルール**：
- Yes が2つ → **あり**（加点要素として +1）
- Yes が1つ以下 → **なし**（加点なし）

### ④ 表現の工夫
- **工夫_Q1**: 黒・鉛筆の灰色以外の色（赤・青・緑・茶色など）が1色以上使われているか
- **工夫_Q2**: Yes/Noの分岐記号、矢印、囲み枠などで構造が視覚的に表現されているか
- **工夫_Q3**: 機器のイラストや絵などの図示があるか

**判定ルール**：
- Yes が2つ以上 → **あり**（加点要素として +1）
- Yes が1つ以下 → **なし**（加点なし）

---

## 加点要素数の計算
- 説明の充実度: 「充実」のみ +1
- 量的充実: 「8割以上」のみ +1
- 思考・考察の記述: 「あり」のみ +1
- 表現の工夫: 「あり」のみ +1

**合計2個以上 → 3点、1個以下 → 2点**

## 各点数の意味
- **3点（良い）**: 量と質の両方に努力が見られる、模範例
- **2点（程よい）**: 取り組んではいるが、量または質のいずれかに明確な不足あり
- **1点（悪い）**: 一般的レベルに達していない、手抜き、量不足

```json
    "量_Q2": "<Yes または No>",

    "工夫_Q3": "<Yes または No>"
  },
  "観点別評価": {

```
